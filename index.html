Weather Report

This web app was created for my sophomore engineering class final project. It uses weatherapi.com to retrieve weather data in the form of a JSON object about a user inputed location. It displays wind, air quality, and forecast information.

Random Word Generator

This was a quick JSFiddle experiment where I generated pseudo words by combining two letter phrases that consisted of a consonant and a vowel, each from two lists. Then, a random number generator chooses from each list to create these two letter phrases, then combines them in a string.

Interactive Makerspace Map

This is an interactive svg map of my school/company's workshop. On hovering over certain parts of the map, information is displayed using tippy.js. Previous members of software development at RCMakes were stuck on this for two years, and I fixed it in three days of work.
